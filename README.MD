## Run Locally

Clone the project

```bash
  git clone https://github.com/AgroFam/AgroFam-FrontEnd
```

Go to the project directory

```bash
  cd AgroFam-FrontEnd
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
